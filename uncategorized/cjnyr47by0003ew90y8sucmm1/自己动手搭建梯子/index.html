作为程序员，拥有稳定的梯子非常重要。至于为什么，代码写多了你自然明白。

###说明

警告：本教程只讨论技术，想用代理赚钱或者做非法勾当的请立刻离开本站。

###可能遇到的问题

  1. 代理开启不成功？可能是端口号被禁用了，通过命令 iptables -A INPUT -p tcp --dport 代理端口号 -j ACCEPT（这里指的代理端口号是指配置文件 shadowsocks.json 中的 server_port ），可以开放端口。
  1. 代理开启了，但是上不了网？可能是你的服务器所处安全组没有将代理端口放通，通过服务器提供商的控制面板编辑安全组规则，将代理端口放通即可。

###客户端
代理开启来了，怎么才能访问 Google 呢？ Android, Mac, Windows 的话推荐 GitHub 上的 ShadowsocksX-NG 客户端，iOS 由于种种问题，推荐使用 Potatso Lite 客户端，完全免费，不过国内的 AppStore 不支持下载，但可以通过一些方法可以下载到，这里不详细说明。

###个人观点

技术本无罪，被坏人利用了，却可以带来可怕的后果。所以有了自己的梯子自己用就行，千万不要去为了那点钱去做非法交易，把自己折腾进去了就真的不值得啦。
